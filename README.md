# Cash Flow Minimizer

A Java Swing desktop application that minimizes the number of cash transactions required to settle debts among multiple participants.

## Features

- Add multiple participants
- Record transactions between participants
- Calculate net balances automatically
- Minimize the number of transactions required for settlement
- Interactive GUI built using Java Swing
- Object-Oriented design

## Technologies Used

- Java
- Java Swing
- OOP
- Collections Framework
- HashMap
- ArrayList

## Project Structure

src/
├── Main.java
└── CashFlowMinimizerGUI.java

## How to Run

1. Clone the repository

```bash
git clone https://github.com/SujalRajput003/Cash-Flow-Minimizer.git
```

